# Welcome to Offensive Tooling for Operators

**Offensive Tooling for Operators** is a 2-day hands-on security course built around a live Active Directory lab environment. Day 1 covers the recon and external testing tools you use to gather intelligence and identify attack surface. Day 2 follows a real attack chain through credential capture, hash cracking, AD enumeration, and lateral movement — ending with domain compromise.
